# hollys-glow-cleaning-website

Premium website for Holly's Glow Suds and Scents L.L.C. in Orlando, FL.

## Tech Stack

- Next.js
- React
- Tailwind CSS

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm run start
```
